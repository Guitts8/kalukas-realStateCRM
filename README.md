Kaluka's CRM

Kaluka's CRM is a real estate management system designed to help agencies organize properties, manage clients, and centralize business operations.

The project is structured with a separated architecture (API, backend, and frontend) to keep the code organized and scalable.

⚠️ The project is currently under development.

Getting Started

Clone the repository:

git clone https://github.com/yourusername/kalukas-crm.git

Navigate to the project folder:

cd kalukas-crm

Then run the backend or frontend according to the project configuration.
